# Установщик 3X-UI

Автоматическая установка панели [3X-UI](https://github.com/MHSanaei/3x-ui) одной командой с SSL-сертификатами и автопродлением.

## 🚀 Установка

Скрипт всё сделает сам — никаких вопросов, просто ждите завершения.

```bash
bash <(curl -Ls https://raw.githubusercontent.com/antonproit/3x-ui-fast-install/main/setup-3xui.sh)
```

## 🛠️ Что устанавливается?

- **OpenSSL**, **QRencode**, **3X-UI**
- SSL-сертификат (6 дней) с автопродлением

## ⚙️ После установки

По окончании скрипт выведет данные для входа в панель, которые вам необходимо сохранить: логин, пароль, порт и ссылку.
Откройте ссылку в браузере и войдите с указанными данными.

Управление панелью:

```bash
x-ui
```

## 📌 Требования

- Ubuntu 20.04 / 22.04 / 24.04
- Root-доступ
